# LUA ULTIMATE INT

Make it possible to calculate large number (bignum) in Lua, Pure Lua number library!\
**POWER BY INT DOT LUA**

Example:

```lua
-- require a module
local int = require("int")

-- build a new int object
local x, y = int.new("20", "10")

print(x + y) -- output: 30
```

## feature

- **Calculate**
  - addition `+`
  - subtraction `-`
  - multiplication `*`
  - division `\, \\`
  - modulo `%`
  - power `^`
  - sqrt
- **Equation**
  - equal
  - less than
  - more than
- **Function**
  - exp
  - fact
  - sqrt
  - *and more...*
